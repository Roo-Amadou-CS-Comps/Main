Implementing, Analyzing, and Defending Against Password Spraying Attacks on HTTP-based Authentication Systems

Roo Case '25, Amadou Touré '25

Project Description

RASpray is a password spraying tool developed to demonstrate the challenges and vulnerabilities inherent in authentication systems.

Over the course of this term, we worked on RASpray, our version of the password-spraying tool Hydra. We submitted our tool as well as a writeup, poster, and video explaining how it works.
